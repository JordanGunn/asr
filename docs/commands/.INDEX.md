# Commands

- [oasr](ROOT.md) - Global flags and options
- [oasr add](ADD.md) - Add a skill to the registry
- [oasr adapter](ADAPTER.md) - Generate IDE/Tooling adapters
- [oasr clean](CLEAN.md) - Clean up the registry
- [oasr help](HELP.md) - Show the help menu
- [oasr find](FIND.md) - Find/discover skills in your file system.
- [oasr info](INFO.md) - Show information about a skill
- [oasr list](LIST.md) - List all registered skills
- [oasr rm](RM.md) - Remove a skill from the registry
- [oasr status](STATUS.md) - Show the status of the registry
- [oasr sync](SYNC.md) - Sync skills with their source locations (local/remote)
- [oasr update](UPDATE.md) - Update the `oasr` CLI.
- [oasr use](USE.md) - Use a skill in any workspace (Open-closed principle)
- [oasr validate](VALIDATE.md) - Validate a skill
